# Soundboard Live

Application web de soundboard pour performance live.

Fonctions principales :

- pads audio avec import de fichiers ;
- enregistrement micro ;
- volume, pan, fade in/out, loop ;
- modes de lecture `1x`, `Hold`, `Toggle` ;
- boards/projets avec pads séparés ;
- stockage local dans le navigateur ;
- installation possible sur iPhone via GitHub Pages en HTTPS.

## Publier avec GitHub Pages

1. Pousser ce dossier dans un repository GitHub.
2. Dans GitHub, ouvrir `Settings > Pages`.
3. Dans `Build and deployment`, choisir `Deploy from a branch`.
4. Branch : `main`, folder : `/root`.
5. Ouvrir l'URL GitHub Pages fournie.

Sur iPhone, ouvrir l'URL HTTPS GitHub Pages dans Safari, puis `Partager > Sur l'ecran d'accueil`.
